# Lisa Ronquillo GitHub Portfolio

A clean, one-page portfolio website for Lisa Ronquillo's GitHub work, designed for professional and academic sharing.

## Files

- `index.html` - semantic page structure and portfolio content
- `styles.css` - responsive light-mode styling
- `script.js` - small enhancements for the footer year and external links

## Run Locally

Open `index.html` directly in a browser, or serve the folder with a simple local server:

```bash
python3 -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## Deploy With GitHub Pages

1. Create or open the GitHub repository for the portfolio.
2. Add `index.html`, `styles.css`, `script.js`, and `README.md` to the repository root.
3. In GitHub, go to **Settings > Pages**.
4. Set the source to the main branch and root folder.
5. Save and wait for GitHub Pages to publish the site.

## Customize

- Update links in the hero and contact sections inside `index.html`.
- Edit project cards in the `Featured Projects` section.
- Adjust the color palette in the `:root` variables at the top of `styles.css`.
- Keep health-related tools framed as educational, reflective, and non-diagnostic.
